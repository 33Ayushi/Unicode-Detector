# Unicode-Detector
The unicode character detector is a free tool that allows you to identify any characters in your text messages that are not part of the GSM character set.
Unicode Detector 🔍

A simple browser-based tool that helps identify Unicode characters in a text message that are not part of the GSM character set.

This tool is useful when working with SMS messages, where non-GSM characters can increase message length or change the encoding to Unicode.

Just paste your text, and the tool will instantly detect characters that are outside the GSM standard.

✨ Features

Detects Unicode characters in text

Identifies characters not in the GSM character set

Instant results in the browser

Simple and user-friendly interface
